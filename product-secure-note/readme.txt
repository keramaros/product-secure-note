== Description ==
This plugin allows store owners to add a customizable security note to each WooCommerce product. Enhance customer trust and ensure a safer shopping experience with tailored security information displayed alongside your products.

== Installation ==
Here’s a step-by-step guide on how to install this WordPress plugin:

= Via the WordPress Dashboard =
1. In your WordPress admin dashboard, go to Plugins > Add New.
2. Click Upload Plugin.
3. Select the downloaded .zip file and click Install Now.
4. Once installed, click Activate.
= Via FTP =
1. Use an FTP client (like FileZilla) to connect to your WordPress site’s server.
2. Navigate to the /wp-content/plugins/ directory.
3. Upload the unzipped plugin folder to this directory.
4. Log in to your WordPress admin dashboard and activate the plugin under Plugins > Installed Plugins.
